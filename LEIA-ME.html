Passo 1 — Baixar o Instalador

Escolha o instalador para o seu sistema operacional:

🍎

macOS

SETUP_..._MAC.command

Clique duplo no Finder

🐧

Linux

SETUP_..._LINUX.sh

Clique duplo ou: bash SETUP...sh

🪟

Windows

SETUP_..._WINDOWS.zip

Extrair zip, clicar no .bat

Passo 2 — Instalar

1

Execute o instalador

Clique duplo no arquivo. O instalador vai abrir e guiar você por todas as etapas.

Na primeira vez, ele instala todas as dependências automaticamente. Pode demorar alguns minutos.

2

Escolha a pasta de instalação

O instalador sugere a pasta padrão de programas do seu sistema. Aceite ou escolha outra.

3

Aguarde a instalação

A barra de progresso mostra o andamento. Na primeira vez baixa o modelo de voz (~460MB).

4

Pronto!

Um ícone será criado na área de trabalho. Clique nele para abrir o programa.

Passo 3 — Configurar o Programa

1

Conecte o celular Android via USB

Ative a Depuração USB nas configurações do desenvolvedor do celular.

Configurações → Sobre o telefone → toque 7x no "Número da versão" → volte → Opções do desenvolvedor → Depuração USB

2

Instale o BCR (Basic Call Recorder)

Instale no celular Android o app BCR para gravação de chamadas.

Disponível no GitHub: github.com/chenxiaolong/BCR

3

Abra o programa e configure

Cole a URL do Google Sheets (acima) no campo indicado.

Dê um nome para o celular (ex: cel1, cel2).

4

Clique em INICIAR

O programa vai pedir números da nuvem, discar automaticamente, classificar e enviar os resultados de volta. Roda sem parar até você clicar em PARAR.

Abas do Google Sheets

Os resultados aparecem organizados nestas abas:

Atendeu — pessoa real atendeu
Tentar Novamente — aguardando religar
Não Atende — tentou 3x, ninguém pegou
Desligados/Inexistentes — bloqueado/fora de área
Descartados — número não existe

Números que não atendem são religados automaticamente até 3 vezes, com intervalo de horas entre cada tentativa. Se em qualquer tentativa a pessoa atender, o número vai para a aba "Atendeu".

Escalar com Múltiplos Celulares

Para usar mais celulares ao mesmo tempo:

1

Instale o programa em cada computador

Cada computador precisa de um celular Android conectado via USB.

2

Use a mesma URL do Google Sheets

Todos usam a mesma URL. O sistema garante que nenhum número é discado por dois celulares.

3

Dê nomes diferentes

Cada celular precisa de um nome único: cel1, cel2, cel3...

Importante: Os números são gerados aleatoriamente nas proporções corretas por operadora (41% Vivo, 25% Claro, 20% TIM, 14% Oi) com DDD 92. O sistema nunca liga para o mesmo número duas vezes.
